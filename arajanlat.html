Nagyker rendelés · Levél

Excel export csak Váci út áruháznál (nagyker). Mentés: asztal Divian Nagyker mappa — ugyanaz a fájlnév esetén felülírás. Budaörs nem kerül az Excelbe.

Rendelés leadás (Nagyker Excel): csak Cyncly tervező import után — hozd át a Terméklista tételeit, add meg a konyha típust, majd exportálj. Tartalom: vasalat, kisker, gép, tálca, csap (bútorlista a tervezőből jön).

I.+II. részlet: konyha neve, vasalatok és plusz tételek név szerint, szállítás — mindegyik 50% I./II. részletre. Olívazöld fejléc, nettó/ÁFA/bruttó. Szállítólevél: konyha név, bútor db, munkalap darab, vevő/cím — Excel → Szállítólevelek Excel, PDF → Mentett megrendelők / vevő mappa (sorszámmal). JSON-ból: a Mentett megrendelők mappában lévő .json (a megrendelő PDF párja) — Excel + PDF egyszerre. Számlázz.hu: Látványterv / fotó sáv — díjbekérő és szállítólevél a helyi szerver (:17321) felé. Alapból helyi bizonylat (Számlázz API nélkül): DB előnézet / SZL előnézet nyomtatható HTML; Díjbekérő / Szállítólevél: helyi PDF mentés (nyomtatás → PDF). Éles Számlázz: SZAMLAZZ_LOCAL_ONLY=0 + szamlazz-agent-key.txt. Szállítólevél száma: éles módban automatikusan beíródik — helyi módban a generált szám csak előnézet.

Kér felmérést: levél Attilának (gombasattila@gmail.com) — csak az ügyfél adatai. Kötelező: név, cím, telefon.

Thunderbird levél: alap szöveg (Kedves …, ajánlat csatolva, 50% előleg, garancia) — összegek nélkül, a PDF-ben vannak az árak. Feladó: fejlécben választható (vaciut@divian.hu vagy diviankonyhamuhelyekvaciut2@gmail.com).
